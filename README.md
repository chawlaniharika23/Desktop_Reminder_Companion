# Desktop Companion

A cute desktop mascot built with Electron and React.

## Features (Planned)

- Animated mascot
- Walks across the desktop
- Water reminders
- Speech bubbles
- Custom reminder intervals
- Settings panel

## Tech Stack

- React
- Electron
- Vite
- JavaScript
